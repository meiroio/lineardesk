# LinearDesk User Guide

LinearDesk gives customers one place to file support requests and follow their
progress without needing Linear access.

## Sign in

- **Web portal** — sign in with Google or request a magic link for your approved work email.
- **Slack** — use the LinearDesk Slack actions if your workspace has the app installed.

## File a request

Use the web portal form when you need to report a bug or support request. Include
what you expected to happen, what happened instead, and any steps that help the
support team reproduce the issue.

If your team uses Slack intake, you can also create requests from `/ticket`, the
"Create LinearDesk ticket" message shortcut, or by mentioning `@LinearDesk` in a
thread.

## Track requests

Your request list shows the tickets available to your signed-in account,
including their current Linear status.

Your request list is shared across your customer organization. Other approved
users from the same organization can see tickets filed under your company's
approved domains, and you can see theirs.

Open a request to see its details and latest status. If editing is available,
use the **Edit** button on an open request to update the title, description, or
severity.
